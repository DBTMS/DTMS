Install Node Agent

Download and install the monitoring agent on your server or device.

2

Configure API Key

Use the generated API key to authenticate your node with the system.

export DTMS_API_KEY="your_api_key_here"
3

Start Monitoring

Start the agent and verify connection in the dashboard.

sudo systemctl start dtms-agent
